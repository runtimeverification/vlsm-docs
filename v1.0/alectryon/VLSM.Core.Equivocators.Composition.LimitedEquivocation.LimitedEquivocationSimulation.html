VLSM Equivocators Simulating limited message equivocation traces

In this module we show that the composition of equivocators with no-message equivocation and limited state-equivocation can simulate all traces with the fixed_limited_equivocation_property.
As a corollary we show that any state which is valid for the composition of regular nodes using a limited_equivocation_constraint can be obtained as the projection of a valid state for the composition of equivocators with no message equivocation and limited state equivocation.
